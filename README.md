# Turing Robot SDK

A PHP SDK for [图灵机器人](www.tuling123.com)

# 安装依赖

```
composer install
```

# 样例

```
require 'vendor/autoload.php';
require 'turingRobot.class.php';

$test = new TuringRobot('你的 API Key','你的 secret');

echo $test->say('你好');
```
